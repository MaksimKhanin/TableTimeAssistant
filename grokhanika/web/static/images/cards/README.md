# Арт карточек

Сюда кладётся арт карточек. Поле `Card.image_id` — это имя файла в этой папке
(например `goblin.png`). Если файл существует — карточка показывает его; если
поля нет или файла нет — фронтенд рисует заглушку по типу карточки (эмодзи +
цвет). Так арт можно добавлять и менять, не трогая данные карточки.

Рекомендуемый формат: квадратные/портретные PNG или JPG, ~400×300+.
