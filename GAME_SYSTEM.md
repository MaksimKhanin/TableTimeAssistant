# TableTime Assistant — Игровая Система

> Упрощённая RPG-система, вдохновлённая D&D, но с собственными правилами.

---

## Характеристики (4 атрибута)

По умолчанию каждый атрибут равен **5**. При создании персонажа доступно **10 свободных очков** для распределения. Минимальное значение атрибута — **1** (можно «продавать» очки из базовых 5, получая дополнительные). Итого сумма всех четырёх атрибутов не превышает **30**.

| Атрибут | Ключ | Базовый бонус | Эффект |
|---------|------|---------------|--------|
| **Сила** | `strength` | `floor(СИЛ / 2)` | +HP, +физический урон |
| **Ловкость** | `dexterity` | `floor(ЛОВ / 2)` | +физическая защита, +бросок физ. атаки; определяет порядок инициативы |
| **Мудрость** | `wisdom` | `floor(МДР / 2)` | +магическая защита, +бросок маг. атаки/заклинания |
| **Харизма** | `charisma` | `floor(ХАР / 2)` | +ментальная защита, +бросок ментальной атаки (убеждение/обман/угрозы) |

## Производные характеристики

| Параметр | Формула | База |
|----------|---------|------|
| **HP** | `10 + floor(СИЛ / 2)` | 10 |
| **Физическая защита** | `floor(ЛОВ / 2)` | 0 |
| **Магическая защита** | `floor(МДР / 2)` | 0 |
| **Ментальная защита** | `5 + floor(ХАР / 2)` | 5 |
| **Физ. урон** | `1d4 + floor(СИЛ / 2)` | 1d4 |
| **Бонус физ. атаки** | `floor(ЛОВ / 2)` | 0 |
| **Бонус маг. атаки** | `floor(МДР / 2)` | 0 |
| **Бонус мент. атаки** | `floor(ХАР / 2)` | 0 |

## Боевые броски

### Физическая атака
`d20 + Бонус физ. атаки` vs **Физическая защита** цели → попадание → урон `1d4 + floor(СИЛ/2)`

### Магическая атака / Заклинание
Заклинания хранятся в БД и имеют поле **Сложность** (целое число).

1. Атакующий: `d20 + floor(МДР/2)` vs **Сложность заклинания** → если не достиг — промах.
2. Если достиг → цель делает **Магический спасбросок**: `d20 + Маг. защита` vs результат броска атакующего.
   - Если спасбросок выше → урон **половинится** (округление вверх).
   - Если спасбросок равен или ниже → полный урон.

### Ментальная атака (убеждение / обман / угроза)
`d20 + Бонус мент. атаки` vs **Ментальная защита** цели.

---

## Инициатива

Персонажи с **более высокой Ловкостью** ходят первыми. При равной Ловкости — бросок `d20` для определения порядка.

---

## Заклинания (Spell Cards)

Хранятся в БД. Каждая карточка заклинания имеет:
- **name** — название
- **description** — описание эффекта
- **difficulty** — сложность броска атаки (`d20 + МДР/2` должен достичь этого числа)
- **damage_dice** — нотация урона, например `2d6`
- **damage_type** — `physical` / `magical` / `mental`

---

## Типы бросков (roll types)

| Код | Описание |
|-----|----------|
| `initiative` | Инициатива: `d20 + ЛОВ` |
| `attack` | Физ. атака: `d20 + Бонус физ. атаки` |
| `mag_attack` | Маг. атака / заклинание: `d20 + Бонус маг. атаки` |
| `mental_attack` | Ментальная атака: `d20 + Бонус мент. атаки` |
| `damage` | Физ. урон: `1d4 + floor(СИЛ/2)` |
| `save_phys` | Физический спасбросок: `d20 + Физ. защита` |
| `save_mag` | Магический спасбросок: `d20 + Маг. защита` |
| `save_mental` | Ментальный спасбросок: `d20 + Мент. защита` |
| `check_str` | Проверка Силы: `d20 + floor(СИЛ/2)` |
| `check_dex` | Проверка Ловкости: `d20 + floor(ЛОВ/2)` |
| `check_wis` | Проверка Мудрости: `d20 + floor(МДР/2)` |
| `check_cha` | Проверка Харизмы: `d20 + floor(ХАР/2)` |

## Пример персонажа

**Варвар Торин** (Человек)  
СИЛ 10 / ЛОВ 7 / МДР 5 / ХАР 8 (сумма = 30, бюджет использован)

- HP: 10 + 5 = **15**
- Физ. защита: 3
- Маг. защита: 2
- Мент. защита: 5 + 4 = **9**
- Физ. урон: `1d4+5`
- Бонус физ. атаки: +3
- Бонус маг. атаки: +2
- Бонус мент. атаки: +4
